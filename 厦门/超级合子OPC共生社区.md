# 超级合子OPC共生社区

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 厦门市思明区 |
| 详细地址 | 思明区软件园二期望海路8号 |
| 揭牌时间 | 2025年 |
| 运营方 | 超级合子、厦门软件园 |
| 定位 | 厦门首个OPC共生社区、AI创业生态 |

## 社区简介

超级合子OPC共生社区是厦门市首个专门面向一人公司（OPC）的创业社区，位于厦门软件园二期核心位置。社区以"共生"为理念，强调创业者之间的协作、资源共享和生态共建。

厦门软件园是中国知名的软件和信息服务业基地，拥有完善的产业链和丰富的技术人才资源。超级合子OPC共生社区依托软件园的优势资源，为AI创业者提供优质的创业环境。

## 入驻政策

### 入驻条件

- 人工智能领域项目
- AI+软件、AI+互联网、AI+跨境电商等方向
- 一人公司或小微创业团队

### 空间补贴

| 支持项目 | 具体内容 |
|----------|----------|
| 工位补贴 | 符合条件的OPC可享受工位补贴 |
| 租金减免 | 入驻初期租金减免政策 |
| 创业补贴 | 厦门市创业补贴政策 |

### 厦门市OPC支持政策

- 厦门市AI产业扶持政策
- 软件园专项政策支持
- 人才引进和住房补贴

## 入驻流程

1. 联系超级合子运营方或厦门软件园
2. 提交项目计划书和入驻申请
3. 项目评审和面试
4. 签订入驻协议
5. 正式入驻

## 配套服务

- **共生社群**：创业者社群活动和交流
- **资源共享**：工具、渠道、人脉共享
- **技术支持**：AI技术培训和咨询
- **融资服务**：创投机构对接
- **软件园资源**：享受软件园各类服务

## 适合人群

- **AI+软件**方向创业者
- **AI+跨境电商**项目（厦门跨境电商发达）
- **AI SaaS**产品开发者
- 希望在**软件园生态**中创业的OPC
- 福建及东南沿海地区创业者

## 厦门创业优势

1. **软件产业**：中国软件名城，产业生态完善
2. **跨境电商**：跨境电商产业发达
3. **对台优势**：对台交流便利，可拓展台湾市场
4. **生活环境**：气候宜人，生活品质高
5. **人才资源**：厦门大学等高校支撑

## 注意事项

1. 厦门首个OPC共生社区，生态理念特色鲜明
2. 位于软件园核心位置，资源丰富
3. 5年培育200家OPC的目标显示发展决心
4. 建议关注厦门软件园最新政策
