# 模数OPC社区（亦庄）

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 北京市经济技术开发区（亦庄） |
| 详细地址 | 通明湖信息城（模数世界新质产业社区） |
| 揭牌时间 | 2026年1月31日 |
| 运营方 | 北京经开区、WaytoAGI |
| 定位 | 北京第二个OPC社区、大模型+数据双核驱动 |

## 社区简介

模数OPC社区是北京经开区（亦庄）打造的人工智能OPC创业社区，依托"模数世界"新质产业社区，以50万平方米宏大规划和"大模型+数据"双核引擎，重新定义AI孵化范式。

国内头部AI社区WaytoAGI已全面入驻并将深度参与社区运营，"数字生命卡兹克"等20余家优质OPC已正式签约落地。

## 入驻政策

### 5A支持体系

| 体系 | 英文 | 内容 |
|------|------|------|
| 工具赋能 | AItools | AI工具和平台支持 |
| 加速成长 | Amplification | 项目加速孵化服务 |
| 协同网络 | Alliance | OPC社群和资源网络 |
| 服务护航 | Assistance | 一站式创业服务 |
| 生态营造 | Atmosphere | 创业氛围和社区文化 |

### 算力与资金支持

| 支持项目 | 具体内容 |
|----------|----------|
| 算力补贴 | 每年投入3亿元发放算力券 |
| 模型补贴 | 最高50%模型使用补贴 |
| 算力折扣 | 最高30%算力补贴 |
| 订单支持 | 真实订单派发 |
| 专项资金 | 最高200万元专项资金支持 |

### 入驻即收益

社区推行"入驻即接单，开工即收益"模式，通过真实产业订单对接，让OPC创业者快速实现商业化。

## 入驻流程

1. 联系模数OPC社区或北京经开区科技部门
2. 提交项目计划书和入驻申请
3. 项目评审和路演
4. 签订入驻协议
5. 正式入驻，享受5A服务

## 配套服务

- **WaytoAGI社区**：国内头部AI社区资源对接
- **算力支持**：算力券、模型券支持
- **订单对接**：真实产业场景和订单
- **创业服务**：一站式创业服务
- **社群网络**：OPC创业者交流社群

## 适合人群

- **AI大模型**方向创业者
- **数据应用**领域OPC
- 希望快速获得**真实订单**的创业者
- 需要**算力支持**的AI项目
- 北京及北方地区AI创业者

## 亦庄创业优势

1. **产业基础**：北京高精尖产业主阵地
2. **政策力度**：年投入3亿元支持OPC
3. **生态资源**：WaytoAGI等头部社区入驻
4. **场景丰富**：真实产业订单对接
5. **交通便利**：地铁亦庄线直达

## 注意事项

1. 北京第二个OPC社区，与中关村AI北纬社区形成南北呼应
2. 强调"入驻即接单"，适合有明确产品方向的OPC
3. WaytoAGI社区资源是重要优势
4. 建议提前准备好项目demo
