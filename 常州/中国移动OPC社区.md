# 中国移动（常州）OPC社区

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 常州市 |
| 启动时间 | 2026年1月（开年首个工作日） |
| 运营方 | 中国移动 |
| 定位 | 常州首批OPC社区 |
| 特色 | 通信资源、5G+AI |

## 社区简介

中国移动（常州）OPC社区是常州市在2026全国智能体开发者大会暨常州市"人工智能+"工作推进会上启动的首批两大OPC社区之一（另一个是硅基智能OPC社区）。

依托中国移动的通信资源和技术能力，为AI领域的一人公司提供5G、算力、数据等基础设施支持。

## 入驻政策

### 常州市OPC政策

| 支持项目 | 具体内容 |
|----------|----------|
| 办公空间 | 租金减免、零租工位 |
| 运营奖励 | 社区运营主体每年最高200万元奖励 |
| 生活配套 | 10分钟办公生活圈 |
| 一站式服务 | 公司注册、安居落户、融资对接 |

### 通信资源优势

- 5G网络支持
- 云计算资源
- 大数据服务
- 物联网能力

## 入驻流程

1. 联系中国移动常州分公司或社区运营方
2. 提交入驻申请
3. 项目评审
4. 签订入驻协议
5. 入驻并接入中国移动资源

## 配套服务

- **通信资源**：5G、专线等网络服务
- **算力支持**：移动云计算资源
- **办公配套**：设施齐全的共享办公空间
- **生活配套**：餐饮、便利店、休闲健身
- **政策辅导**：各类补贴申请服务

## 适合人群

- **5G+AI**应用方向创业者
- 需要**通信资源**支持的项目
- 物联网、智慧城市相关项目
- 需要**稳定网络基础设施**的OPC

## 注意事项

1. 中国移动背景，通信资源丰富
2. 适合需要5G、云计算等基础设施的项目
3. 常州新兴OPC生态，政策支持力度大
4. 具体入驻条件需咨询社区运营方
