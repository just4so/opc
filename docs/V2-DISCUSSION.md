## 广场 /plaza(✅ 方向已确认)

### 整体重新设计

**三个视图 Tab:**
- **人**(默认):以人为主体,卡片显示头像/名字/认证/bio/城市/方向 + 关联产品(产品名+tagline+阶段)
- **产品**:以产品为主体,卡片显示产品名/tagline/阶段 + 创始人信息
- **动态**:保留现有帖子列表

**筛选栏(三个视图通用):**
- 方向、城市、阶段 + 搜索框(人名/产品名/关键词)

**顶部引导条:**
- 没有卡片:"创建你的名片,让 XX 位创业者看到你" → /settings#card
- 有卡片没产品:"添加你的产品,让更多人找到你" → /settings#card
- 有卡片有产品:"编辑我的信息 →"

### 砂掉 lookingFor/canOffer
- 数据库字段保留,前端不再展示和收集
- 卡片内容围绕"人+产品"设计,不再用抽象标签

### 直通车→广场联动(核心设计)

**直通车表单重新设计:**
```
第一步:基本信息
- 姓名
- 微信(联系方式)
- 城市
- 意向社区(combobox · 城市)

第二步:关于你和你的产品
- 一句话介绍自己(bio)
- 产品/服务名称
- 一句话介绍产品(tagline)
- 产品阶段(想法/开发中/已上线/有收入/已盈利)
- 产品网站(选填)
- 上传 BP 或公司介绍(选填,提示提高推荐成功率)

☑ 同时展示在创业者广场(默认勾选)
```

**提交后自动完成:**
1. 创建 Inquiry 记录(直通车留资)
2. 回写用户 profile(bio、城市)
3. 自动创建 Project(产品名、tagline、阶段、网站)
4. showInPlaza = true(如勾选)
5. 广场自动多一张卡片

**效果:转化漏斗和内容增长变成同一件事。**

### 体验优化(推演发现的断点)
- 设置保存后加"去广场看看你的卡片 →"引导
- 未登录点"联系TA"→注册后回跳到当前 profile 页
- 卡片/profile 加"最近活跃"时间
- 后续加"本周新入驻""热门创业者"等变化内容(P2)

## 用户主页 /profile/[username](✅ 已确认)

**页面结构(上到下):**
1. **头部:** 头像 + 名字 + 认证badge + 城市 + 方向 + 阶段 + 最近活跃时间 + bio + 「联系TA」「分享」按钮
2. **产品区块:** 卡片式展示每个产品(名称 + tagline + 阶段 + 网站链接)
3. **最近动态:** 只显示最近 2-3 条帖子标题,弱化展示
4. **完善度(仅自己可见):** 进度条 + 缺失项提示 + "去完善 →"

**设计原则:**
- 头部 3 秒定生死,一眼看完"这人是谁做什么的"
- 产品是主角,比动态更大更靠前
- "最近活跃"解决"这个人还在不在"的疑虑
- lookingFor/canOffer 不再展示
