- [ ] 搜索词 > 100 字 → 后端 400 提示

## 自动化测试状态

| 层级 | 工具 | 覆盖 | 运行命令 |
|------|------|------|---------|
| 类型检查 | TypeScript | 全量 | `npx tsc --noEmit` |
| 单元测试 | Vitest | 校验逻辑/业务规则 | `npm test` |
| E2E 测试 | Playwright | 公开页面/导航/权限 | `npm run e2e` |
| CI | GitHub Actions | tsc + prisma + vitest | 自动（push/PR）|
