## 一、页面总览

### 主导航（3项）
| 导航 | 页面 | 说明 |
|------|------|------|
| 找社区 | `/communities` | 社区列表 + 地图，底部有"申请收录"入口 |
| 创业者广场 | `/plaza` | 双 Tab：创业者卡片墙 + 动态帖子 |
| OPC雷达 | `/radar` | 不动，保持原样 |

### 右上角
- 未登录：「登录」+「创建卡片」
- 已登录：🔔 通知铃铛 + 头像下拉菜单

### 隐藏页面（不在导航，通过入口或直接URL访问）
| 页面 | 说明 |
|------|------|
| `/news` | 创业资讯（首页底部有入口） |
| `/tools` | 工具导航（首页底部有入口） |
| `/connect` | 通用直通车（登录后访问） |
| `/connect/[slug]` | 指定社区直通车 |
| `/search` | 全站搜索 |
