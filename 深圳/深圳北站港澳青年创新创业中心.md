# 深圳北站港澳青年创新创业中心

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 深圳市龙华区 |
| 详细地址 | 深圳北站附近 |
| 定位 | 深圳首批OPC社区之一 |
| 特色 | 港澳青年创业支持 |

## 社区简介

深圳北站港澳青年创新创业中心是深圳龙华区首批OPC社区之一，专项支持港澳青年创业。位于深圳北站附近，交通便利，可快速往返香港，是港澳青年内地创业的理想选择。

## 入驻政策

### 深圳市级政策

| 优惠项目 | 具体内容 |
|----------|----------|
| 办公空间 | 最大50平方米低成本创业空间 |
| 智能券 | 零门槛发放 |
| 训力券 | 最高1000万元 |
| 模型券 | 最高200万元 |
| 语料券 | 最高200万元 |

### 港澳青年专项

- 港澳青年创业专项扶持
- 深港两地通勤便利
- 港澳青年人才政策

## 入驻流程

1. 联系龙华区或创业中心运营方
2. 提交入驻申请（港澳青年优先）
3. 项目评审
4. 签订入驻协议
5. 正式入驻

## 配套服务

- 港澳青年创业辅导
- 深港资源对接
- 交通便利（深圳北站）
- 人才服务

## 适合人群

- **港澳青年**创业者
- 需要**深港两地**资源的项目
- 重视**交通便利**的创业者

## 注意事项

1. 优先支持港澳青年
2. 深圳北站交通枢纽，通勤便利
3. 可享受港澳青年专项政策
