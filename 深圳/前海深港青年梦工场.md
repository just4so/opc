# 前海深港青年梦工场

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 深圳市前海 |
| 详细地址 | 前海深港合作区 |
| 定位 | 专项支持港澳青年创业的OPC社区 |
| 特色 | 深港合作、跨境优势 |

## 社区简介

前海深港青年梦工场是深圳首批OPC社区之一，专项支持港澳青年创业。前海作为深港合作的桥头堡，拥有独特的政策优势和跨境便利条件，是港澳青年内地创业的首选地。

社区聚焦跨境业务，为做跨境出海方向的OPC创业者提供得天独厚的地理和政策优势。

## 入驻政策

### 深圳市级政策

| 优惠项目 | 具体内容 |
|----------|----------|
| 办公空间 | 最大50平方米低成本创业空间 |
| 智能券 | 零门槛发放 |
| 训力券 | 最高1000万元 |
| 模型券 | 最高200万元 |
| 语料券 | 最高200万元 |

### 前海特色政策

- 港澳青年专项扶持政策
- 跨境业务便利化
- 深港合作优惠政策
- 15%企业所得税优惠（符合条件的企业）

### 人才政策

- 博士、硕士、本科分档最高10万元入户补贴
- 最长36个月过渡性住房
- 港澳青年专项人才支持

## 入驻流程

1. 联系前海深港青年梦工场
2. 提交入驻申请（港澳青年优先）
3. 项目评审
4. 签订入驻协议
5. 入驻并享受配套服务

## 配套服务

- **跨境便利**：深港通关便利、跨境金融服务
- **港澳资源**：香港高校、投资机构对接
- **政策支持**：前海自贸区政策优惠
- **创业辅导**：针对港澳青年的创业培训

## 适合人群

- **港澳青年**创业者（优先支持）
- **跨境电商/出海**方向创业者
- 需要**深港两地资源**的项目
- 希望利用**自贸区政策**的企业

## 注意事项

1. 优先支持港澳青年创业项目
2. 前海有独特的自贸区税收优惠政策
3. 跨境业务需了解相关合规要求
4. 建议提前了解港澳青年创业专项政策
