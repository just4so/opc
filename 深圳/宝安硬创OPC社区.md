# 宝安硬创OPC社区

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 深圳市宝安区 |
| 定位 | 全国首个面向OPC硬创业的系统性产业支撑体系 |
| 特色 | 场景+金融+制造 |

## 社区简介

宝安区押注硬件创业，以"场景+金融+制造"构建全国首个面向OPC硬创业的系统性产业支撑体系。依托区内5.8万家制造业企业资源，为AI硬件创业者提供从设计、打样到批量生产的全链条服务。

宝安区是深圳传统制造业强区，拥有完善的电子制造产业链，是硬件创业的理想之地。

## 入驻政策

### 深圳市级政策

| 优惠项目 | 具体内容 |
|----------|----------|
| 办公空间 | 最大50平方米低成本创业空间 |
| 智能券 | 零门槛发放 |
| 训力券 | 最高1000万元 |
| 模型券 | 最高200万元 |
| 语料券 | 最高200万元 |

### 宝安区特色支持

- **制造资源**：5.8万家制造业企业对接
- **供应链**：电子元器件全链条支持
- **金融支持**：硬件创业专项金融服务
- **场景开放**：真实产业场景验证

## 入驻流程

1. 联系宝安区科创部门或社区运营方
2. 提交硬件创业项目申请
3. 项目评审
4. 签订入驻协议
5. 入驻并对接制造资源

## 配套服务

- **5.8万家制造企业**资源对接
- 从设计到生产的**全链条服务**
- 硬件**快速打样**
- **小批量生产**支持
- 产业**场景验证**

## 适合人群

- **AI硬件**创业者
- 智能设备/消费电子创业者
- 需要**快速量产能力**的项目
- IoT/嵌入式系统创业者

## 注意事项

1. 宝安区聚焦硬件创业，纯软件项目不适合
2. 制造业资源丰富，适合有量产需求的项目
3. 建议提前准备产品设计方案或原型
4. 可充分利用宝安区供应链优势降低成本
