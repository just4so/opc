# 鸿鹄汇（上城区）

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 杭州市上城区 |
| 详细地址 | 杭州东站旁融信中心13层 |
| 空间面积 | 1200平方米，42个工位 |
| 创始人 | 邹凌 |
| 定位 | 国内首家主打"一人创业"的加速社区 |

## 社区简介

鸿鹄汇（Honghub）是国内首个专为AI时代"一人公司"提供创业加速的社区平台，于2025年9月在融信中心启用。社区重点聚焦**AI生产力工具**与**AI创意内容**两大前沿赛道。

短短3个月已收到超过500份申请，最终吸引27位顶尖个体入驻，皆为海归技术专家、名校博士及头部科技企业资深开发者，其中还包括4名外籍人士。42个工位已有三分之二坐满。

## 入驻政策

### 核心福利

| 优惠项目 | 具体内容 |
|----------|----------|
| 启动资金 | **5万美元**无附加条件启动资金 |
| 办公空间 | **完全免费**（上城区5万平方米免租计划） |
| 装修费用 | 政府承担 |
| 水电网络 | 闸弄口街道承担 |

### 配套政策

- 引才资助
- 研发补助
- 贷款贴息
- 融资奖励

### 上城区OPC政策

- 5万平方米免租计划
- 开放湖滨、吴山、钱江新城三大商圈作为产品"试炼场"
- 上城资本集团百亿AI母基金，专项规划首期1亿元OPC子基金
- 目标：2026年集聚30家以上AI重点企业，引进培育300名以上OPC青年人才

## 入驻条件

鸿鹄汇筛选标准（较高门槛）：
1. **规模小**：一人或极小团队
2. **AI能力强**：能充分借力AI，让一个人发挥一个团队的作用
3. 具备全栈开发能力、跨领域整合经验

## 入驻流程

1. 提交项目申请（通过官方渠道）
2. 项目评审筛选（通过率约5%）
3. 签约入驻
4. 领取工位，接入社区资源
5. 享受5万美元启动资金等支持

## 配套服务

- **产品打磨期**：5万美元资金支持
- **商业化前期**：帮助对接场景
- **发展稳定期**：协助连接投资与资源
- **三大商圈**：湖滨、吴山、钱江新城产品试炼场

## 适合人群

- 具备**全栈开发能力**的独立开发者
- 海归技术专家、名校博士
- 头部科技企业出来的资深开发者
- AI生产力工具/AI创意内容方向创业者

## 注意事项

1. **筛选门槛较高**：500份申请仅入驻27位
2. 重点聚焦AI生产力工具和AI创意内容赛道
3. 需要有一定技术实力和项目成熟度
4. 5万美元启动资金无附加条件，不占股
