# 魔方空间

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 无锡市新吴区（高新区） |
| 详细地址 | 无锡国家软件园内 |
| 揭牌时间 | 2025年12月8日 |
| 定位 | 首批OPC社区 |
| 特色 | 软件园生态、AI+软件 |

## 社区简介

魔方空间位于无锡国家软件园内，是无锡高新区（新吴区）首批揭牌的OPC社区之一。依托无锡国家软件园的产业生态，为AI+软件领域的一人公司提供创业服务。

无锡高新区同步发布了支持OPC发展的"十条政策"，从产业要素、研发创新、科技载体、人才保障四个方面全面支持OPC创新业态发展。

## 入驻政策

### 无锡高新区OPC十条政策

| 支持方向 | 具体内容 |
|----------|----------|
| 产业要素 | 算力、数据等资源支持 |
| 研发创新 | 研发补贴、技术支持 |
| 科技载体 | 低成本办公空间 |
| 人才保障 | 人才补贴、住房支持 |

### 发展目标（到2028年）

- 培育OPC企业 **100家以上**
- 集聚AI创客人才 **1000名**

## 入驻流程

1. 联系魔方空间或无锡国家软件园招商部门
2. 提交入驻申请材料
3. 项目评审筛选
4. 签订入驻协议
5. 正式入驻、接入园区资源

## 配套服务

- **软件园生态**：无锡国家软件园产业资源
- **技术支持**：软件开发技术服务
- **办公配套**：共享办公空间
- **政策服务**：各类补贴申请辅导
- **产业对接**：软件园企业资源对接

## 适合人群

- **AI+软件**领域创业者
- 软件开发、SaaS产品方向
- 需要**软件园产业资源**的OPC
- 无锡本地或愿意扎根无锡的创业者

## 注意事项

1. 位于无锡国家软件园内，产业氛围浓厚
2. 适合软件、AI应用等技术方向项目
3. 可享受无锡高新区OPC专项政策
4. 需关注具体入驻条件和申请时间
