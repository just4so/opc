# 亲橙OPC社区（阿里首个OPC社区）

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 南京市建邺区 |
| 详细地址 | 河西中央科创区，南京阿里中心T5栋 |
| 空间规模 | 首期508个工位 |
| 运营模式 | 阿里巴巴+政府共建 |
| 定位 | 阿里巴巴集团旗下首个OPC社区、江苏省规模最大OPC社区 |

## 社区简介

2025年12月12日，建邺区河西中央科创区亲橙OPC社区正式揭牌运营。这是**阿里巴巴集团旗下首个OPC社区**，也是**江苏省规模最大的OPC社区**。

项目以人工智能为主要产业发展方向，采用"总部大厂牵头、政府政策赋能、市场机制驱动"的全新共建模式。阿里巴巴将开放其技术平台、应用场景和产业生态；建邺区则聚焦于精准的政策供给、高效的政务服务和优越的产业环境。

目前已集聚30余个创业团队，众多个人创业者得以"拎包入驻"，当天开工。

## 入驻政策

### "六个一"专属权益包

| 权益项目 | 具体内容 |
|----------|----------|
| 一个社区主理人 | 专属服务管家 |
| 一个智能化平台 | 阿里云生态+魔搭开发者社区 |
| 一个集成服务站 | 一站式政务服务 |
| 一个实战训练营 | 创业培训支持 |
| 一个共享交流社 | 社区交流平台 |
| 一套专项政策包 | 载体、模型、算力、数据、应用、人才全要素支持 |

### 南京市政策叠加

- 省、市、区**三级政策叠加**支持
- 阿里生态资源接入
- 云上平台远程算力、协同工具一键调用

## 入驻流程

1. 联系河西中央科创区或南京阿里中心
2. 提交入驻申请
3. 项目评审（面向高校"创新火种"、大厂"创业精英"、行业"极客先锋"）
4. 签订入驻协议
5. 拎包入驻，当天开工

## 配套服务

- **阿里云生态**：技术平台、应用场景接入
- **魔搭社区**：开发者资源
- **云上平台**：远程算力、协同工具
- **政务服务**：一站式集成服务站
- **培训支持**：实战训练营

## 适合人群

- 高校**"创新火种"**：高校背景创业者
- 大厂**"创业精英"**：互联网大厂出来的创业者
- 行业**"极客先锋"**：技术极客创业者
- 希望接入**阿里生态**的创业者

## 注意事项

1. 这是阿里首个OPC社区，阿里资源对接优势明显
2. 河西中央科创区位置优越
3. 可享受省市区三级政策叠加
4. 未来计划在区域内布局OPC社区专属载体超5万方
