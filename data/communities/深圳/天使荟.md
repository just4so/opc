# 天使荟（福田区）

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 深圳市福田区 |
| 详细地址 | 福田区（具体地址请咨询福田区科创局） |
| 定位 | 深圳首批OPC社区之一 |

## 社区简介

天使荟是深圳市福田区首批OPC社区之一，聚焦AI+硬件等四大核心赛道。福田区依托华强北"一米柜台"升级为AI时代轻量化创业载体，为OPC创业者提供独特的创业环境。

福田区以"场景城区"战略突围，开放金融、医疗等海量场景，发布首批"创新机会清单"41个项目，为OPC创业者提供真实的商业落地机会。

## 入驻政策

### 深圳市级政策

| 优惠项目 | 具体内容 |
|----------|----------|
| 办公空间 | 最大50平方米低成本创业空间 |
| 智能券 | 零门槛发放 |
| 训力券 | 最高1000万元 |
| 模型券 | 最高200万元 |
| 语料券 | 最高200万元 |

### 人才政策

- 博士、硕士、本科分档最高10万元入户补贴
- 最长36个月过渡性住房（市场价60%租金）
- 或最长24个月、每月1250元安居补贴

### 福田区特色

- 华强北供应链资源对接
- 金融、医疗等场景开放
- "创新机会清单"项目对接

## 入驻流程

1. 咨询福田区科创局或天使荟运营方
2. 提交入驻申请及项目介绍
3. 通过评审
4. 签订入驻协议
5. 正式入驻

## 配套服务

- **场景资源**：金融、医疗等真实商业场景
- **供应链支持**：华强北硬件供应链
- **创新机会清单**：41个项目需求对接
- **政策辅导**：各类补贴申请服务

## 适合人群

- AI+金融方向创业者
- AI+医疗/健康方向创业者
- AI+硬件创业者
- 需要场景验证的AI项目

## 注意事项

1. 福田区聚焦四大核心赛道，建议选择匹配方向
2. 华强北资源适合硬件相关项目
3. 金融、医疗场景有合规要求，需提前了解
4. 需注册企业后入驻
