# 模力营（南山区）

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 深圳市南山区 |
| 详细地址 | 南山智城片区（云科技大厦、神州数码国际创新中心） |
| 运营主体 | 力合科创 |
| 联系人 | 力合科创创新部副总经理 贺飞 |

## 社区简介

"模力营"是深圳南山区打造的AI生态社区，依托大湾区最大人工智能垂直孵化加速器，面积超10万平方米。社区一期位于云科技大厦和神州数码国际创新中心，周围环绕鹏城实验室、深圳科创学院、英特尔创新中心、开源中国社区等优质资源。

2025年6月3日正式开营，首批12家人工智能企业入驻。目前已有近200家优质企业入驻，区内人工智能企业数量占全市比例近4成。

## 入驻政策

### 空间补贴

| 优惠项目 | 具体内容 |
|----------|----------|
| 租金优惠 | **两年免租** |
| 空间规模 | 计划提供十万平方米产业空间 |

### 算力补贴

- 每年最高支持可达**100万元**算力补贴

### 政策券支持

| 券类型 | 最高额度 |
|--------|----------|
| 训力券 | 1000万元 |
| 模型券 | 200万元 |
| 语料券 | 200万元 |
| 智能券 | 根据企业规模发放 |

### 综合政策

- 市区两级政府"拎包入住"政策
- 南山区AI十条专项政策
- 总额超2000万元资金补助

## 入驻流程

1. 联系力合科创或南山区相关部门咨询
2. 提交入驻申请和项目介绍
3. 评审考察（重点考察AI领域相关性和发展潜力）
4. 通过审核后签订入驻协议
5. 入驻办公空间，享受配套服务

## 配套服务

- **场景应用支持**：通过场景揭榜挂帅机制，在真实环境中验证AI解决方案
- **孵化加速服务**：百家创新主体孵化，千项智能应用催生
- **产业资源对接**：鹏城实验室、高校、英特尔创新中心等资源
- **政策申报辅导**：各类补贴政策申请指导

## 适合人群

- AI大模型领域创业者
- 人工智能应用开发者
- AI+行业解决方案提供者
- 需要算力支持的AI项目团队

## 注意事项

1. 重点支持人工智能领域项目
2. 需注册为企业（个体工商户或一人有限公司）才能入驻
3. 各类补贴券需通过申请审核后发放
4. 建议提前准备好营业执照和项目商业计划书
