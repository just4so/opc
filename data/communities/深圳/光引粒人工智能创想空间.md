# 光引粒·人工智能创想空间

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 深圳市光明区 |
| 定位 | 深圳首批OPC社区之一 |
| 特色 | 算力供需桥梁 |

## 社区简介

光引粒·人工智能创想空间是深圳光明区首批OPC社区，主要功能是搭建算力供需桥梁，为AI创业者提供算力资源对接服务。

光明区作为深圳的科学城核心区，拥有大科学装置和科研资源，为AI创业提供强大的技术支撑。

## 入驻政策

### 深圳市级政策

| 优惠项目 | 具体内容 |
|----------|----------|
| 办公空间 | 最大50平方米低成本创业空间 |
| 智能券 | 零门槛发放 |
| 训力券 | 最高1000万元 |
| 模型券 | 最高200万元 |
| 语料券 | 最高200万元 |

### 算力支持

- 算力供需对接平台
- 优惠算力资源
- 训练任务支持

## 入驻流程

1. 联系光明区科创部门或社区运营方
2. 提交入驻申请
3. 项目评审
4. 签订入驻协议
5. 入驻并接入算力资源

## 配套服务

- **算力资源**：搭建算力供需桥梁
- **科研资源**：光明科学城大科学装置
- **技术支持**：AI开发技术指导
- **政策辅导**：各类补贴申请服务

## 适合人群

- 需要大量**算力资源**的AI创业者
- AI模型训练方向项目
- 需要**科研资源**支持的项目
- 深度学习、大模型相关创业者

## 注意事项

1. 光明区重点支持需要算力的AI项目
2. 可对接光明科学城科研资源
3. 位置相对偏远，需考虑通勤因素
4. 适合重算力需求的项目
