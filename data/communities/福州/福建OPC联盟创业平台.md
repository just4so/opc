# 福建OPC联盟·仓山AI创业助手平台

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 福州市仓山区 |
| 详细地址 | 仓山区互联网小镇（具体地址待确认） |
| 揭牌时间 | 规划中（2026年） |
| 运营方 | 福建OPC联盟、仓山区相关部门 |
| 定位 | 福建省OPC联盟示范基地、AI创业服务平台 |

## 社区简介

福建OPC联盟·仓山AI创业助手平台是福建省响应国家支持一人公司和AI创业政策，规划建设的省级OPC联盟示范项目。平台依托仓山区互联网小镇的产业基础，打造面向全省OPC创业者的服务平台。

福建作为数字中国建设的先行省份，正在积极推动AI产业发展和一人公司创业支持。福建OPC联盟旨在整合全省资源，为OPC创业者提供系统性支持。

## 发展规划

| 阶段 | 目标 |
|------|------|
| 2026年 | 完成基础设施建设，启动首批入驻 |
| 2027年 | 服务100+OPC企业 |
| 2028年 | 建成省级OPC服务标杆 |

## 入驻政策（规划中）

### 入驻条件

- 人工智能领域项目
- AI+数字经济、AI+制造、AI+服务等方向
- 福建省内及外省优秀OPC项目

### 支持政策（预期）

| 支持项目 | 具体内容 |
|----------|----------|
| 空间补贴 | 工位租金补贴 |
| 创业补贴 | 福建省/福州市创业补贴 |
| 算力支持 | 福建省算力资源对接 |
| 政策服务 | 政策申报一站式服务 |

## 配套服务（规划中）

- **联盟资源**：福建OPC联盟成员资源共享
- **创业指导**：创业导师和专家咨询
- **融资服务**：对接福建省内创投机构
- **产业对接**：福建省产业资源对接
- **政策服务**：各类补贴政策申报

## 适合人群

- **福建籍**AI创业者
- 希望拓展**福建市场**的OPC
- **AI+传统产业**转型方向
- 对福建省政策和资源感兴趣的创业者

## 福州创业优势

1. **数字福建**：数字中国建设先行省份
2. **海西经济区**：连接台湾和东南亚
3. **生活成本**：相比一线城市成本较低
4. **政策支持**：福建省积极支持AI和数字经济
5. **产业基础**：互联网小镇等数字产业基础

## 注意事项

3. 可先关注福建省科技厅和福州市科技局政策
