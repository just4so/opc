# 原点空间（西太湖人工智能国际社区）

## 基本信息

| 项目 | 内容 |
|------|------|
| 所在城市 | 常州市武进区西太湖 |
| 全称 | 西太湖（常州）人工智能国际社区"原点空间" |
| 定位 | 人工智能OPC创业社区 |
| 特色 | 国际化、AI产业"超级连接器" |

## 社区简介

原点空间是西太湖（常州）人工智能国际社区的核心载体，定位为AI产业的"超级连接器"。已开展多场主题沙龙活动，正从起步探索迈向体系化服务阶段。

常州《关于加快集聚人工智能领域人才的若干举措》明确以西太湖（常州）人工智能国际社区为重点，推动建设一批高品质、国际化人工智能OPC创业社区。

## 入驻政策

### 常州市OPC政策

| 支持项目 | 具体内容 |
|----------|----------|
| 办公空间 | 拎包入驻、租金减免、零租工位 |
| 运营奖励 | 社区运营主体每年最高200万元奖励 |
| 生活配套 | 10分钟办公生活圈 |
| 一站式服务 | 公司注册、安居落户、融资对接、研发创新 |

### 国际化特色

- 国际人才引进政策
- 跨境资源对接
- 国际化社区氛围

## 入驻流程

1. 联系西太湖人工智能国际社区或武进区相关部门
2. 提交入驻申请
3. 项目评审
4. 签订入驻协议
5. 拎包入驻

## 配套服务

- **产业连接**：AI产业"超级连接器"
- **主题沙龙**：定期举办行业交流活动
- **国际资源**：国际化资源对接
- **生活配套**：餐饮、便利店、休闲健身
- **全流程服务**：覆盖全天候、贯穿全链条

## 适合人群

- 有**国际化**背景或需求的AI创业者
- 需要**产业资源对接**的OPC
- 希望参与**行业交流**的创业者
- 看好常州AI产业发展的创业者

## 注意事项

1. 位于西太湖，环境优美但位置相对偏远
2. 国际化定位，适合有跨境需求的项目
3. 常州重点打造的AI社区
4. 定期有主题沙龙，社区氛围活跃
